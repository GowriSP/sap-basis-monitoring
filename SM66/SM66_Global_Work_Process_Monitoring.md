# SM66 Global Work Process Monitoring

## Purpose

SM66 is used to monitor all work processes across all application servers in the SAP system.

## Transaction Code

SM66

## Features

- Monitor all active work processes
- Identify long-running processes
- Analyze system-wide performance issues
- Check CPU and memory-consuming tasks
- Monitor user sessions

## Activities Performed

1. Execute transaction SM66
2. Review active work processes
3. Identify processes with high runtime
4. Analyze user and transaction details
5. Take corrective action if required

## Common Issues

### Long Running Dialog Process
- Check user and transaction
- Analyze SQL execution
- Contact functional team if required

### High CPU Consumption
- Identify affected work process
- Analyze system load
- Check database performance

## Benefits

- Real-time monitoring
- Performance troubleshooting
- System health analysis

## Related Transactions

- SM50 – Local Work Process Monitoring
- ST06 – OS Monitoring
- ST22 – Dump Analysis
- DBACOCKPIT – Database Monitoring

## Result

Helps BASIS administrators identify and resolve system performance issues proactively.
